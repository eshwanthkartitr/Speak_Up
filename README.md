# Speak Up 

Despite the increasing global awareness around inclusively, millions of individuals who communicate using sign language still face barriers in education, employment, and everyday interaction. Traditional sign language learning tools often lack accessibility, interactivity, and personalization, making it difficult for beginners and non-native learners to master communication skills effectively.

Using speak up we are planning on making sign language learnable by anyone
